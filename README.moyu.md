## ⚡ 怎么用（30 秒版）

```bash
# 趁老板没注意赶紧 clone
git clone https://github.com/peng/retire-buddy.git

# 假装在 npm install 生产环境依赖
pnpm install

# 启动！快！老板来了切到 VS Code！
pnpm dev
```

> 💡 小贴士：`pnpm dev` 启动后浏览器看起来很像一个正经的内部管理系统，放心摸。
